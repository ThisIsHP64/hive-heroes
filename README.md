# Hive Heroes - SER225 RPG Game

## What is this?
This is a game created for Quinnipiac's SER225 Agile-focused Software Development class. This is a 2D RPG that revolves around the player being the savior of a dying hive.

## How to run this project?
No external libraries were used for this, so you can clone this repo as is and run it out of the box with Java (minimum version 8).<br><br>
An IDE such as IntelliJ or Eclipse is recommended to run this game.

The main method for the game is in the `Game` package > `Game.java` file.<br><br>
The main method for the map editor is in the `MapEditor` package > `MapEditor.java` file.
